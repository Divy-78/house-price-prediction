# House Price Prediction

This is a supervised learning machine learning project that predicts house prices using Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Workflow
1. Data Cleaning
2. Feature Encoding
3. Train-Test Split
4. Model Training
5. Prediction
6. Evaluation

## Model Used
- Linear Regression
